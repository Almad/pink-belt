
========
Glossary
========

Terms used in this documentation.

.. glossary::

	Story Owner
		Person responsible for a specific user story that's going to be developed.

	Developer
		Person transmuting a wish into the code that's running in production. 

	Product Board
		Trello board with User Story cards. 

	Story
	Story Card
	User Story
	User Story Card
		Single Trello card on :term:`Product Board` that describes a single user story.

	Work Board
	Work Overview
		Trello board with deployeable tasks/units derived from User Story cards.
